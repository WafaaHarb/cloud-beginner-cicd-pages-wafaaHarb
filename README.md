## Project Description
This project demonstrates a complete CI/CD pipeline using GitHub Actions to automatically deploy a static website to GitHub Pages.

## Local Development
To run this site locally:
1. Clone the repository
2. Open `index.html` in your browser
3. No build tools or servers required!

## CI/CD Pipeline
When code is pushed to the `main` branch:
1. GitHub Actions automatically triggers the workflow
2. Basic file checks verify all required files exist
3. Site is deployed to GitHub Pages
4. Live site updates within 1-2 minutes
